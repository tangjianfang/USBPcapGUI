安装说明
  1. 访问 nodejs.org，下载 LTS 版本的 Windows 安装包（.msi）。
  2. 运行安装程序，按照默认选项完成安装。
  3. 打开命令提示符（CMD），运行 node --version 确认版本号 ≥ 20。
  4. 安装完成后，回到项目目录运行 cd gui && npm install && npm start 启动 Web GUI。

安装说明
  1. 从官方网站下载最新版 USBPcap 安装程序。
  2. 以管理员身份运行安装程序。
  3. 安装完成后可能需要重启电脑。
  4. 安装后在本页面点击"重新检测"按钮确认状态。

🚀 快速启动

安装 Node.js

2
启动 Web GUI 
cd gui
